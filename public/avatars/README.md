## Especificações recomendadas

- **Formato:** PNG (preferencialmente quadrado, com transparência se for retrato sobre fundo)
- **Tamanho:** 256×256 px (suficiente, fica nítido em qualquer tela)
- **Aspecto:** quadrado (1:1) — o sistema arredonda automaticamente como círculo

## Como funciona

Se um arquivo não existir aqui, o usuário verá um círculo colorido com a inicial do personagem como fallback — funcional, só sem o retrato. Adicione os arquivos quando tiver e eles aparecerão automaticamente sem precisar mexer no código.
